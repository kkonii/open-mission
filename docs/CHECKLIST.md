## 기능 요구사항

- `한국어`, `일본어`, `영어` 의 버전으로 안내 문구 언어를 설정할 수 있다.


- 각 자동차의 기종을 선택하고, 이름을 부여할 수 있다. 기종과 이름을 `-`로 연결하여 입력한다
    - `BIKE`, `CAR`, `FERRARI`, `TAXI`, `BUS` 의 기종이 있다.
    - 이름은 5자 이하만 가능하다.
- 같은 기종의 자동차는 중복해서 선택할 수 없다.
- `기종-이름`의 짝은 `,`로 연결해서 입력한다.


- 기종마다 각각 다른 전진 조건을 가지고 있다. `0-10` 까지의
    - `BIKE`: 랜덤 숫자 생성 후 4 이하이면 1칸 전진
    - `CAR`:  랜덤 숫자 하나를 생성 후, 3일 때만 1칸 전진
    - `FERRARI`: 랜덤 숫자가 8 이상이면, 2칸 전진
    - `TAXI`: 랜덤 두 개를 생성해서, 첫 번째 숫자가 두 번째 숫자보다 커야만 1칸 전진
    - `BUS`: 랜덤 숫자가 홀수이면 1칸 전진


- 사용자는 배팅할 횟수(N)를 입력할 수 있다.
- 매 회차당 우승할 자동차를 예측하여 입력해야 한다


- 한 회차의 ROUND 수는 사용자가 입력하지 않는다.
- 한 경기는 항상 고정된 라운드만큼 도는 것으로 한다.


- 자동차 경주 게임을 완료한 후, 이동거리가 긴 자동차부터 순위별로 출력한다
- 동순위가 발생하면 같은 등수를 부여하고, 다음 등수는 ‘동순위 인원 수만큼’ 건너뛴다. 예시로, 1등인 자동차가 두 명이면 그 다음은 3위부터 출력한다
- 동순위 자동차의 이름은 쉼표(,)를 이용하여 구분한다.


- 한 회차가 끝날 때마다 배팅의 결과를 알려준다
- 모든 회차의 경주가 종료되면 오늘의 승률을 출력한다
- 승률은 `예측이 성공한 경기 수 / 배팅한 경기 수 * 100` 로 계산한다


- 사용자가 잘못된 값을 입력할 경우 `IllegalArgumentException`을 발생시키고, 다시 입력받는다

<br>

## 입출력 요구사항

#### 입력

- 경주할 자동차 기종과 이름을 `-`로 연결하여 입력
- 참여자 이름은 `,`로 연결하여 입력

```text
FERRARI-경연,TAXI-우테코,BUS-포비,CAR-돌고래
```

- 배팅할 경주 횟수

```text
3
```

- 배팅할 자동차 입력 (우승 예상자)

```text
우테코
```

#### 출력

- 경주 회차별 결과

```text
1회차 경기 결과

          0    5    10   15   20
          
경연    |  ██▮
우테코  | 
포비    |  ██▮
돌고래  |  ████▮
```

- 회차별 순위 발표

```text
1등   |  돌고래
2등   |  경연, 포비
4등   |  우테코
```

- 회차별 배팅 결과 (예측한 우승자와 실제 우승자가 다를 경우)

```text
1회차 경기 우승자 예측에 실패했습니다.
```

- 최종 승률 발표

```text
승률은 25.6% 입니다.
```

## 프로그래밍 요구 사항

- [ ] 메서드 라인은 `10줄`을 넘기지 않는다
- [ ] depth 는 `1`까지만 허용한다
- [ ] 스트림 체이닝은 3단계를 넘어가지 않도록 구현한다. 3줄이 넘으면 리팩터링을 고려한다

허용 범위 예시

```java
  return products.stream()
      .

filter(this::method1)
      .

map(this::method2)
      .

toList();
```

- [ ] 가능한 ArrayList<> 가 아닌 다른 자료 구조를 활용하여 구현한다
- [ ] 3항 연산자를 쓰지 않는다
- [ ] `camp.nextstep.edu.missionutils` 에서 제공한 api 는 직접 구현하여 사용한다
- [ ] 정규식을 사용하지 않고 검증을 구현해본다

- [ ] service, repository 를 도입해서 구조를 설계한다
- [ ] 입출력 기능에 대해 테스트 코드를 작한다

## 환경 요구 사항

- [ ] 이어폰이나 귀마개를 끼지 않고 (소음을 차단하지 않고) 진행한다
- [ ] 유동 인구가 있는 외부 공간에서 진행한다
