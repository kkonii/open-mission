## 🖥️ 개발 기능 목록

안내 문구 기능

- [ ] 입력받을 때 차량의 종류를 출력한다
- [ ] 형식은 `[차량1,차량2,차량3]` 와 같이 출력한다

입력 기능

- [x] 자동차 차량과 이름을 입력받는다

- [x] 시도 횟수를 입력받는다
- [ ] 시도 횟수를 숫자 타입으로 변환한다
- [ ] **<예외>** 숫자가 아닌 경우, 예외를 발생한다

- [ ] **<공통 예외>** 이름과 시도횟수가 한 개도 입력되지 않으면(공백) 예외를 발생한다

---

파싱 기능

- [ ] 입력값을 구분자(`,`) 로 자른다
- [ ] `차량-이름`을 구분자(`-`)로 자른다

검증 기능

- [ ] **<예외>** 두개의 입력값이 지정된 구분자(-)로 연결되지 않으면 예외를 발생한다
- [ ] **<예외>** `차량-이름` 입력값들이 지정된 구분자(,)로 연결되지 않고 입력되면 에외를 발생한다

---

경주 기능

1. 시도 횟수

- [ ] **<예외>** 시도 횟수가 0 이하인 경우, 예외를 발생한다

2. 자동차 기종

- [x] 입력받은 이름과 일치하는 기종을 찾는다

- [x] **<예외>** 제시된 기종이 아니면 예외를 발생한다
- [ ] **<예외>** 중복된 기종이 입력된 경우, 예외를 발생한다

3. 자동차 이름

- [ ] **<예외>** 중복되는 이름이 존재할 경우 예외를 발생한다
- [ ] 각 자동차에 이름을 부여한다
- [ ] **<예외>** 이름이 `5`자를 초과하면 예외를 발생한다
- [ ] **<예외>** 이름이 `공백`이면 예외를 발생한다
- [ ] **<예외>** 이름이 `한글`,`영어`,`숫자` 외의 문자를 포함하면 예외를 발생한다 (띄어쓰기는 허용)

4. 랜덤 숫자 생성

- [ ] 0-10 중 랜덤 정수 한 개를 반환한다

5. 전진

- [ ] 반환된 숫자가 전진 가능한 범위인지 판단한다
- [ ] 전진 가능한 경우, 이동 거리만큼 움직인다
- [ ] 입력된 횟수만큼 모든 자동차들이 전진 혹은 멈춘다

6. 우승자 판별

- [ ] 자동차들 중 가장 멀리 이동한 거리를 찾는다
- [ ] 가장 멀리 이동한 자동차들을 찾는다

---

출력 기능

1. 차수별 실행 결과

- [ ] 입력한 이름으로 자동차를 출력한다
- [ ] 전진 횟수는 `-`로 표시한다

2. 최종 우승자 안내 문구

- [ ] 참여한 자동차 이름을 출력한다
- [ ] 공동 우승자는 `,` 로 합산하여 표시한다
