# 🚗 자동차 경주 배팅 게임

**<이용 가이드>**

안녕하세요! <br>
자동차 기종을 직접 고르고, 우승자를 예측해 배팅하는 자동차 경주 배팅 게임입니다.
간단한 입력만으로 흥미로운 경주 결과와 승률까지 확인할 수 있어요.
---

## 🌍 지원 언어

게임 실행 시 다음 언어 중 하나를 선택할 수 있습니다.

- 🇰🇷 한국어
- 🇺🇸 English
- 🇯🇵 日本語

---

## 🎮 게임 진행 방식

### 1. 언어 선택

실행 후 원하는 언어 번호를 입력하면 모든 안내 문구가 해당 언어로 출력됩니다.

---

### 2. 자동차 등록

경주에 참가할 자동차들을 등록합니다.  
각 자동차는 다음 두 요소로 구성됩니다:

1. 기종 (`BIKE`/`CAR`/`FERRARI`/`TAXI`/`BUS` 중 하나)
2. 이름 (5자 이하)

#### 입력 형식

```text
기종-이름,기종-이름,기종-이름...
```

예시:

```text
FERRARI-경연,TAXI-우테코,BUS-포비,CAR-돌고래
```

⚠️ 주의

- 기종은 중복 사용할 수 없습니다.
- 이름은 반드시 5자 이하입니다.

---

### 3. 배팅할 경기 횟수 입력

오늘 몇 번의 경기를 배팅할지 정합니다.

예시:

```text
3
```

---

### 4. 회차별 우승자 예측

한 경기 안에서 여러 라운드가 진행된 뒤 우승자가 정해집니다.
경주 시작 전, 여러분은 우승할 것 같은 자동차 이름을 예측하여 입력하면 됩니다.

예시:

```text
우테코
```

---

## ⚙️ 자동차 기종별 전진 규칙

각 기종은 **0~10 사이 랜덤 숫자**로 다음 조건에 따라 전진합니다.

| 기종          | 전진 조건                  |
|-------------|------------------------|
| **BIKE**    | 랜덤 숫자가 `4 이하` → 1칸 전진  |
| **CAR**     | 랜덤 숫자 `3일 때만` → 3칸 전진  |
| **FERRARI** | 랜덤 숫자가 `7 이상` → 1칸 전진  |
| **TAXI**    | 랜덤 숫자가 `홀수` → 1칸 전진    |
| **BUS**     | 랜덤 숫자가 `3의 배수` → 1칸 전진 |

## 🏁 경주 결과 확인하기

경주가 끝나면 다음과 같은 정보가 출력됩니다:

### ✓ 라운드별 이동 결과

```text
=============================
        1회차 경기 결과
=============================
          0    5    10   15
          
경연    |  ██▮
우테코  | 
포비    |  ██▮
돌고래  |  ████▮

```

---

### ✓ 회차별 순위 발표

동순위가 있을 경우 같은 등수를 주고, 다음 등수는 건너뜁니다.

```text
=============================
       1회차 경기 순위 발표
=============================      

1등   |  돌고래
2등   |  경연, 포비
4등   |  우테코

=============================
```

---

### ✓ 배팅 결과

예측에 성공했는지 안내해 줍니다.

예시:

```text
최종 우승자는 [돌고래] 입니다.
1회차 경기 배팅에 실패했습니다.
```

예측 성공 시:

```text
🔴축하합니다! 우승자를 맞췄습니다!🔴
```

---

📊 최종 승률 발표

모든 경기가 종료되면 오늘의 승률을 계산해 보여줍니다.

승률 계산식 =
`예측 성공 점수 / 배팅한 횟수 × 100`

예측 성공 점수는 `1 / 우승자 수`입니다.  (예: 우승자가 2명이면 0.5점)

출력 예시:

```text
=============================
        최종 승률 발표
=============================
최종 승률은 25.6% 입니다.
```

---

## ❗ 입력 오류 처리

잘못된 값을 입력하면 다음처럼 안내합니다:

- 잘못된 기종 이름
- 이름 5자 초과
- 중복된 기종
- 존재하지 않는 자동차에 배팅 시도

이런 경우, 예외 메시지를 출력 후 다시 입력을 받습니다.
