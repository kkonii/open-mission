## 🖥️ 개발 기능 목록

언어 선택 기능

- [x] 선택 가능한 언어를 안내한다
- [x] 언어를 입력받는다
- [x] 출력될 문구를 선택한 언어로 전환한다
- [ ] 예외 문구도 선택한 언어에 맞게 변환한다

- [x] **<예외>** 제공되지 않은 언어를 선택할 경우 예외를 발생한다

안내 문구 기능

- [x] 입력받을 때 차량의 종류를 출력한다
- [x] 형식은 `<차량1,차량2,차량3>` 와 같이 출력한다

입력 기능

- [x] 자동차 차량과 이름을 입력받는다

- [x] 배팅할 경주 횟수를 입력받는다
- [x] 배팅할 경주 횟수를 숫자 타입으로 변환한다
- [x] **<예외>** 숫자가 아닌 경우, 예외를 발생한다

- [x] 예측할 우승 자동차를 입력받는다

- [x] **<공통 예외>** 이름과 시도횟수가 한 개도 입력되지 않으면(공백) 예외를 발생한다

---

파싱 기능

- [x] 입력값을 구분자(`,`) 로 자른다
- [x] `차량-이름`을 구분자(`-`)로 자른다

검증 기능

- [x] **<예외>** 두개의 입력값이 지정된 구분자(-)로 연결되지 않으면 예외를 발생한다
- [ ] **<예외>** `차량-이름` 입력값들이 지정된 구분자(,)로 연결되지 않고 입력되면 에외를 발생한다

---

경주 기능

1. 배팅 횟수

- [x] 최소 횟수는 `2번`으로 제한한다
- [x] 최대 횟수는 `10번`으로 제한한다

- [ ] **<예외>** 배팅 횟수가 0 이하인 경우, 예외를 발생한다
- [x] **<예외>** 제한 횟수를 넘어가는 경우, 예외를 발생한다

2. 자동차 기종

- [x] 입력받은 이름과 일치하는 기종을 찾는다

- [x] 자동차에 차량기종을 부여한다
- [x] **<예외>** 제시된 기종이 아니면 예외를 발생한다
- [x] **<예외>** 중복된 기종이 입력된 경우, 예외를 발생한다

3. 자동차 이름

- [x] **<예외>** 중복되는 이름이 존재할 경우 예외를 발생한다
- [x] 각 자동차에 이름을 부여한다
- [x] **<예외>** 이름이 `5`자를 초과하면 예외를 발생한다
- [x] **<예외>** 이름이 `공백`이면 예외를 발생한다
- [x] **<예외>** 이름이 `한글`,`영어`,`숫자` 외의 문자를 포함하면 예외를 발생한다 (띄어쓰기는 허용)

4. 예측하는 우승자 이름

- [x] **<예외>** 입력한 우승자가 `기능 목록 3`에서 입력한 이름들에 없을 경우, 예외를 발생한다

5. 랜덤 숫자 생성

- [x] 0-10 중 랜덤 정수 한 개를 반환한다

6. 전진

- [x] 반환된 숫자가 전진 가능한 범위인지 판단한다
- [x] 전진 가능한 경우, 이동 거리만큼 움직인다
- [x] 한 회차의 경주당 모든 자동차들이 전진 혹은 멈춘다
- [x] 모든 회차의 경주 라운드는 고정된 숫자로 동일하게 진행한다

7. 우승자 판별

- [x] 자동차들을 거리순으로 내림차순 정렬한다
- [x] 거리에 따른 순위를 계산한다
- [x] 순위가 가장 높은 자동차들을 찾는다

8. 배팅 결과 계산

- [x] 예측한 우승자와 일치하는지 판단한다

9. 경주 진행

- [x] 한 라운드가 끝나면 자동차들의 이동 거리를 초기화한다
- [x] 입력받은 횟수만큼 배팅을 진행한다

10. 최종 승률 계산

- [x] 예측이 일치한 경기수와 총 배팅한 경기수를 이용해 승률을 계산한다

---

출력 기능

0. 공통 포맷팅

- [ ] `이름   |   거리` , `순위  |  이름` 형식으로 출력한다
- [ ] 간격을 일정하게 포맷팅한다

1. 차수별 실행 결과

- [x] 입력한 이름으로 자동차를 출력한다
- [ ] 전진 횟수는 `█`로 표시한다
- [ ] 현재 자동차의 위치를 `▮` 로 표시한다

2. 최종 순위 안내 문구

- [x] 경주 결과를 순위별로 출력한다
- [X] 참여한 자동차 이름을 출력한다
- [X] 동순위 자동차는 `,` 로 합산하여 표시한다

3. 우승자 예측 배팅 결과

- [x] 결과를 출력한다

4. 최종 승률

- [x] 소수점 둘째 자리에서 반올림한다
- [x] `%`를 붙여 포맷팅하여 출력한다

재입력 요청 기능

- [x] 예외 발생시 문구를 출력한다
- [x] 다시 입력을 요쳥한다
